## Description

MoovySpace is a platform for cinephiles to log movies they've watched, star movies that they particularly liked, and see what movies other users have liked in a live feed.

## Technologies

-   HTML5
-   CSS3
-   JavaScript
-   Node.js
-   MongoDB
-   GraphQL
-   Apollo Server
-   MUI
-   React.js
-   MERN
-   Heroku
-   Framer-Motion

No installation required
